# GB28181 P4b — Media-State Layer Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Expose which GB28181 streams are live (and their RTP receive detail) via a `MediaCache` fed by ZLM's `on_media_changed` hook, a `GET /gb/streams` API, and a dashboard status indicator.

**Architecture:** A `MediaCache` (set of live `(app, stream)` keys) owned by `GbBridge` is updated synchronously by a new `on_media_changed` ZLM hook (Regist→insert, UnRegist→remove). `GbBridge::stream_status()` joins the stream mapping with the cache (live flag) and, for live streams, enriches with `ZlmControl::rtp_info`. `GET /gb/streams` serves it; the Vue dashboard polls it to show a per-gb-device live indicator.

**Tech Stack:** Rust (edition 2024), tokio, axum, rszlm ≥ `adff0a1`, Vue 3 + PrimeVue.

**Spec:** `docs/superpowers/specs/2026-07-02-gb28181-zlm-control-layer-design.md` — this is P4b = §8 "media-state layer" (§4.2 MediaCache, §4.6 GET /gb/streams, §4.7 hook). Builds on P4a (`ZlmControl`, `Mapping.transport`, `StreamMap.list`, `ZlmControl::rtp_info`).

**Scope note:** The spec's §4.4 "optional post-INVITE rtp_info verify + retry" is **deferred** (out of this plan) — it interacts with the pre-existing double-pull window (`bridge.rs` TODO) and is best done as a focused follow-up. P4b delivers the *observable* state layer only; `UnRegist` does NOT drive teardown (spec §3.4).

```rust
//! Live-stream registry fed by ZLM's `on_media_changed` hook. A stream key
//! `(app, stream)` is present iff ZLM currently has it registered (publishing).
//! Cheap-clone (Arc) so the hook and the API can share one instance.

use std::collections::HashSet;
use std::sync::{Arc, Mutex};

#[derive(Clone, Default)]
pub struct MediaCache {
    live: Arc<Mutex<HashSet<(String, String)>>>,
}

impl MediaCache {
    pub fn new() -> Self {
        Self::default()
    }

    /// ZLM `on_media_changed` Regist: the stream is now publishing.
    pub fn on_regist(&self, app: &str, stream: &str) {
        self.live
            .lock()
            .unwrap()
            .insert((app.to_string(), stream.to_string()));
    }

    /// ZLM `on_media_changed` UnRegist: the stream stopped.
    pub fn on_unregist(&self, app: &str, stream: &str) {
        self.live
            .lock()
            .unwrap()
            .remove(&(app.to_string(), stream.to_string()));
    }

    pub fn is_live(&self, app: &str, stream: &str) -> bool {
        self.live
            .lock()
            .unwrap()
            .contains(&(app.to_string(), stream.to_string()))
    }

    pub fn live_streams(&self) -> Vec<(String, String)> {
        self.live.lock().unwrap().iter().cloned().collect()
    }
}

#[cfg(test)]
#[path = "media_cache_test.rs"]
mod media_cache_test;
```

```rust
    pub async fn stream_status(&self) -> Vec<GbStreamStatus> {
        let mappings = self.streams.list();
        let mut out = Vec::with_capacity(mappings.len());
        for (stream_id, m) in mappings {
            let live = self.media_cache.is_live("rtp", &stream_id);
            let rtp = if live {
                self.control.rtp_info("rtp", &stream_id).await

                device_id: m.device_id,
                channel_id: m.channel_id,
                transport: m.transport,
                live,
                rtp,
            });

    }
```

```rust
#[derive(Serialize)]
struct StreamStatusDto {
    stream_id: String,
    device_id: String,
    channel_id: String,
    transport: String,
    live: bool,
    rtp: Option<RtpInfoDto>,
}

#[derive(Serialize)]
struct RtpInfoDto {
    exist: bool,
    peer_ip: String,
    peer_port: u16,
    local_port: u16,
    identifier: String,
}

fn transport_str(t: gb28181::Transport) -> &'static str {
    match t {
        gb28181::Transport::Udp => "udp",
        gb28181::Transport::TcpPassive => "tcp_passive",
        gb28181::Transport::TcpActive => "tcp_active",
    }
}

/// Live status of every gb stream mapping (empty list when GB is disabled).
async fn streams() -> ApiJsonResult<Vec<StreamStatusDto>> {
    let Some(bridge) = crate::gb::bridge() else {
        return Ok(ok_json(Vec::new()));
    };
    let items = bridge
        .stream_status()
        .await
        .into_iter()
        .map(|s| StreamStatusDto {
            stream_id: s.stream_id,
            device_id: s.device_id,
            channel_id: s.channel_id,
            transport: transport_str(s.transport).to_string(),
            live: s.live,
            rtp: s.rtp.map(|r| RtpInfoDto {
                exist: r.exist,
                peer_ip: r.peer_ip,
                peer_port: r.peer_port,
                local_port: r.local_port,
                identifier: r.identifier,
            }),
        })
        .collect();
    Ok(ok_json(items))
}
```
